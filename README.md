# Customer-Support-RAG

An AI-powered customer support chatbot designed to handle user queries about X (Twitter). It uses Mistral-7B with a RAG (Retrieval-Augmented Generation) pipeline to provide accurate, context-aware answers and suggest related questions.

### Features

Customer support Q&A in conversational form

Suggests related questions to guide users

Knowledge-based responses using document retrieval

### Tech Stack

LLM: Mistral-7B-OpenOrca

Backend: Flask

Frontend: React

Vector DB: ChromaDB

Embeddings: all-MiniLM-L6-v2

Hosting/Models: Hugging Face

### Model

Open-Orca / Mistral-7B-OpenOrca

### Coverage Areas

The chatbot answers FAQs related to:

Account login, password, and security

Username, email, and phone settings

Account preferences and notifications

Verified, suspended, and deactivated accounts

Posting, media, hashtags, and engagement

Direct messages and device support

Privacy, safety, and abuse reporting

Platform rules, policies, and guidelines

Creator tools and monetization
